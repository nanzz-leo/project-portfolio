# Project Phase-1: Portfolio Webpage

Welcome to the **Project Phase-1**, a personal portfolio website created with HTML (with CSS in `misty.css`). The project demonstrates fundamental front-end skills and is structured into sections that present the developer, show projects, and link to social profiles.

Key Features Include:
- **matrix-style visual effect** built with raw HTML
- navigation bar with links to different page sections
- Two professional profile highlights: Ethical Hacker & Web Developer
- Responsive layout using div-based section splitting
- Projects section
- Social media integration (LinkedIn, GitHub, Outlook)
- Resume download option
